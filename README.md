# Tourism Management System

The Tourism Management System is a full-stack web application designed to simplify travel planning and management. The system allows users to browse tourism destinations, view tour packages, and manage travel-related information through a user-friendly interface.

The application consists of a frontend built using HTML, CSS, and JavaScript, and a backend developed with Node.js and Express.js. MongoDB is used as the database to store and manage tourism data.

## Features

* View available tourism destinations
* Explore tour packages
* Manage travel information
* Backend API for handling tourism data
* Responsive web interface

## Tech Stack

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Node.js
* Express.js

Database:

* MongoDB (Mongoose)

## Project Structure

* `frontend/` – User interface
* `backend/` – Server-side API and database connection

## Installation

1. Clone the repository
2. Install dependencies using `npm install`
3. Run the backend server
4. Open the frontend in the browser

## Purpose

This project was developed as a full-stack practice project to demonstrate skills in frontend development, backend API creation, and database integration.
